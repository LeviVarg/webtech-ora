HTML Tags

<h1>
The <h1> to <h6> HTML elements represent six levels of section headings. By default, all heading elements create a block-level box in the layout, starting on a new line and taking up the full width available in their containing block.

<a>
The <a> HTML element (or anchor element), with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.

<img>
The <img> HTML element embeds an image into the document.
